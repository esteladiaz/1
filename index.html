DIVs

Div is a block.

Inline means within text.

For a div, height is always collapsed to the smallest of its contents. This is so it can adjust to what one puts in it, and from what a user is viewing it though.

Width is 100% by default. Websites will scroll down, so height can be challenging.

Span tags are inline, can be used to capitalize a single letter at the start of a paragraph.

Bolds have their own tag. B for bold, I for italics, but these are not used anymore. Separation of concerns and 508 appliance.

Tags are about structure, CSS is about style.

One exception not using B or I tags: I = icon now. If there is an icon, you will see I tag.

IDs are strong for specificity. Navs, main content, about pages.
